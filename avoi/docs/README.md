# AVOI — Autonomous AI Agent Platform

AVOI is a self-improving AI agent that lives on your server, remembers what it
learns, and gets more capable the longer it runs.

## Quick Start

```bash
git clone git@github.com:avoi-ai/avoi-agent.git
cd avoi-agent
pip install -e ".[all]"
avoi setup
avoi
```

## Features

- **Self-improving** — Creates skills from experience, refines them over time
- **Persistent memory** — Remembers across sessions, builds a model of you
- **Multi-platform** — CLI, Telegram, Discord, Slack, WhatsApp, Signal, Email
- **Any LLM** — OpenRouter (200+ models), OpenAI, Anthropic, or self-hosted
- **Scheduled tasks** — Natural language cron for reports, backups, automations
- **MCP support** — Connect to any MCP server for extended capabilities
- **40+ built-in tools** — File management, web, code execution, and more

## Configuration

After installation, run `avoi setup` to configure:
- Your LLM provider and model
- Messaging platform connections
- Tool preferences
